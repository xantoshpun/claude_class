# Extended Thinking

Extended Thinking gives Claude a dedicated space to reason deeply before responding. It's like giving Claude a scratch pad — it thinks through the problem thoroughly, then provides a polished final answer.

## When to Use Extended Thinking

- Complex math and logic problems
- Multi-step reasoning chains
- Strategic planning
- Hard coding problems
- Nuanced analysis requiring many considerations

## Basic Usage

```python
import anthropic

client = anthropic.Anthropic()

response = client.messages.create(
    model="claude-opus-4-5",
    max_tokens=16000,
    thinking={
        "type": "enabled",
        "budget_tokens": 10000  # tokens for thinking
    },
    messages=[{
        "role": "user",
        "content": "What is the most efficient algorithm for finding all prime numbers up to 1 billion? Consider time and space complexity tradeoffs."
    }]
)

for block in response.content:
    if block.type == "thinking":
        print("Claude's reasoning:")
        print(block.thinking)
    elif block.type == "text":
        print("\nFinal answer:")
        print(block.text)
```

## Budget Tokens

The `budget_tokens` parameter controls how much Claude can "think":

| Budget | Use For |
|---|---|
| 1,000-5,000 | Moderate reasoning tasks |
| 5,000-10,000 | Complex multi-step problems |
| 10,000+ | Very hard problems (math olympiad, research-level) |

Note: Thinking tokens count toward your total token usage.

## Streaming with Extended Thinking

```python
with client.messages.stream(
    model="claude-opus-4-5",
    max_tokens=16000,
    thinking={"type": "enabled", "budget_tokens": 8000},
    messages=[{"role": "user", "content": "Your complex question"}]
) as stream:
    for event in stream:
        if hasattr(event, 'type'):
            if event.type == 'content_block_start':
                if hasattr(event.content_block, 'type'):
                    if event.content_block.type == 'thinking':
                        print("Thinking...")
```

## Key Takeaways

- Extended Thinking gives Claude a scratch pad for deep reasoning
- Set `budget_tokens` to control how much thinking is allowed
- Thinking content is transparent — you can read Claude's reasoning
- Best for complex problems that benefit from careful step-by-step analysis

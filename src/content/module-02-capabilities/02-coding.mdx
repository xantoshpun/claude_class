# Coding with Claude

Claude is an exceptional coding partner. It understands dozens of programming languages, can debug complex issues, explain unfamiliar code, and write production-quality implementations from descriptions.

## What Claude Can Help With

### Writing Code
Generate functions, classes, modules, or entire files from natural language descriptions.

### Debugging
Paste an error message or broken code — Claude diagnoses the issue and explains the fix.

### Code Review
Get feedback on readability, performance, security vulnerabilities, and best practices.

### Explanation
Understand unfamiliar code, legacy systems, or complex algorithms.

### Refactoring
Modernize, simplify, or optimize existing code without changing behavior.

## Languages Claude Handles Well

Python, JavaScript/TypeScript, Rust, Go, Java, C/C++, SQL, Bash, Ruby, PHP, Swift, Kotlin — and many more.

## Tips for Better Code Results

1. **Specify the language and version** — "Python 3.11", "TypeScript 5", "Node 20"
2. **Describe the interface** — what inputs/outputs, what errors to handle
3. **Share context** — what framework, what library versions, what constraints
4. **Ask for tests** — "also write unit tests using pytest"
5. **Iterate** — "now optimize it for performance" or "add error handling for X"

## Key Takeaways

- Claude supports all major programming languages
- Use it for writing, debugging, reviewing, explaining, and refactoring
- Provide context (language, framework, constraints) for the best output

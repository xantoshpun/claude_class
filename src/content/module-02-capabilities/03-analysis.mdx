# Analysis & Research with Claude

Claude can process and analyze large amounts of information with impressive accuracy. From summarizing research papers to extracting structured data from unstructured text, Claude handles complex analytical tasks.

## Analytical Capabilities

### Document Analysis
Feed Claude entire PDFs, articles, or reports and ask it to:
- Summarize key findings
- Extract specific data points
- Identify themes and patterns
- Compare multiple documents

### Data Analysis
Provide CSV data or tables and ask Claude to:
- Find trends and outliers
- Generate insights
- Write analysis reports
- Suggest follow-up questions

### Research Assistance
Claude can help you:
- Synthesize information from multiple sources
- Identify gaps in reasoning
- Critique arguments
- Suggest related topics to explore

## Example: Summarize a Research Paper

```
Here is a research paper on transformer architectures. Please:
1. Summarize the key contributions in 3 bullet points
2. Explain the methodology in simple terms
3. List any limitations the authors acknowledge

[paste paper content]
```

## Example: Extract Structured Data

```
Extract all company names, founding years, and revenue figures from this article.
Return the data as a JSON array.

[paste article]
```

## Example: Compare Sources

```
Here are two analyses of the same market trend. 
Where do they agree? Where do they conflict? 
Which seems more credible and why?

[paste both sources]
```

## Working with Long Documents

Claude's 200K token context window lets you analyze very long documents in a single request:
- Entire books
- Large codebases  
- Legal contracts
- Annual reports

## Tips for Better Analysis

1. **Be specific** — "What are the three most important risk factors?" is better than "analyze this"
2. **Ask for structure** — "Return your findings as a numbered list"
3. **Request reasoning** — "Explain why you concluded X"
4. **Chain questions** — start broad, then drill down with follow-up prompts

## Key Takeaways

- Claude can analyze documents, data, and research with high accuracy
- Use the 200K context window for very long documents
- Specific, structured requests yield the most useful analysis

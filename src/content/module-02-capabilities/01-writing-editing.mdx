# Writing & Editing with Claude

Claude is one of the most capable writing assistants available. Whether you need to draft content from scratch, improve existing writing, or adapt text for different audiences, Claude handles it with nuance and style.

## What Claude Can Write

- **Professional communication** — emails, reports, proposals, presentations
- **Creative content** — stories, scripts, poetry, marketing copy
- **Technical writing** — documentation, API references, README files
- **Academic writing** — essays, literature reviews, research summaries
- **Social media** — tweets, LinkedIn posts, captions

## Editing & Improving Text

Claude excels at editing in several modes:

### Proofreading
Fix grammar, spelling, and punctuation while preserving your voice.

### Rewriting
Completely restructure content while keeping the core message.

### Tone adjustment
Make text more formal, casual, persuasive, or concise.

### Summarization
Compress long documents into clear, accurate summaries.

## Example Prompt Patterns

**Draft from scratch:**
```
Write a professional email to a client explaining a project delay.
Be apologetic but confident, and include a new timeline.
```

**Improve existing text:**
```
Rewrite the following paragraph to be more concise and persuasive:
[paste your text]
```

**Change tone:**
```
Make this technical explanation accessible to a non-technical audience:
[paste your text]
```

## Tips for Best Results

1. **Be specific about tone** — "formal", "conversational", "urgent"
2. **Provide context** — who is the audience? What is the goal?
3. **Show examples** — "write something like this: [example]"
4. **Iterate** — ask Claude to revise: "make it shorter" or "add more detail to the second paragraph"

## Key Takeaways

- Claude handles all types of writing: professional, creative, technical, and academic
- Editing capabilities include proofreading, rewriting, tone adjustment, and summarization
- Specificity in your prompts leads to much better output
